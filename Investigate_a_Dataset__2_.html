Introduction¶

Analysing no show dataset which includes information about 100K medical appoints in brazil and focuses on whether the patients showed for there appointment or not. The dataset includes the dates and times of the appointment and also the gneder , scholarship , the neighborhood of the hospital and some diseases like Diabetes,Alcoholism,Handcap and Hipertension. the dependent variable here is the No-show colomn which speciefies whether the patient arrived or not. the questions will be all about the prediction of whether the patient will show or not.

Conclusions¶

our work here was devided into three stages¶

introduction:¶

in this part i imported the pandas,numby and matplotlib to the project and described the dataset and what the questions will be about in this project.

data wrangling:¶

in this part i discovered the dataset the datatyes of each column and if there was any null values and then i cleaned the dataset by doing some changes to the dataset.

Exploratory Data Analysis:¶

in this part i asked some questions and answered them using some analysis and plots and discovered that if the hour after 20 the person won't show up and if the person is diseased there is a bigger chance to show up than when he is not diseased.

limitations¶

we are working with sample data which leads to uncertainity ,we find issues with the sample of data that we are working with like missing observations and data that appears to be inconsistent. we have not done any statistical tests in this analysis, we can not imply any statistical conclusions as well as we can not imply any causation from our correlation findings.
